dotfiles
============

Install with

    cd $HOME
    wget https://raw.githubusercontent.com/uberspot/dotfiles/master/.install.sh -O temp_install.sh
    chmod +x temp_install.sh
    ./temp_install.sh
    rm temp_install.sh

GTK theme: Vertex
Icons theme: Vertex
